# HR Аналитика (Docker Project)

Проект состоит из двух Docker-контейнеров: генератора данных и аналитика.

## Как запустить

1. Соберите образы:
   ./run.sh build_generator
   ./run.sh build_reporter

2. Сгенерируйте данные (появятся в папке data/):
   ./run.sh run_generator

3. Сформируйте HTML-отчет (появится в папке data/):
   ./run.sh run_reporter

## Инструкция по просмотру отчета (для проверки в GitHub Codespaces)

Чтобы открыть сгенерированный `report.html` прямо из Codespaces, выполните следующие шаги:

1. Запустите локальный веб-сервер командой:
   ./run.sh report_server
2. В нижней панели Codespaces (где находится терминал) перейдите на вкладку **Ports** (Порты).
3. Найдите в списке порт **8080** (Local Address).
4. Нажмите на иконку глобуса или кнопку **Open in Browser** (Открыть в браузере) в строке с этим портом.
5. Отчет откроется в новой вкладке вашего браузера!
